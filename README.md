### Deutsch ###
Dieses Projekt entstand im Rahmen einer Anwendungsaufgabe während meines Bachelorstudiums.

Die Funktionalitäten beschränken sich auf das Anlegen, Lesen, Verändern und Löschen von Projekten und Mitarbeitern zu den Projekten (CRUD). 

Mitarbeiter haben, anders als Admins eingeschränkte Möglichkeiten im Umgang mit Projekten.

Die Datenbanke ist eine SQLite DB.

Gearbeitet wurde mit Python Flask


### English ###

This project was developed as part of an applied assignment during my bachelor’s degree.

The functionality is limited to creating, reading, updating, and deleting projects and assigning employees to these projects (CRUD).

Employees have, unlike admins, restricted permissions when working with projects.

The database is a SQLite database.

The project was built using Python Flask.
